My Project

This is a web-based Stock Market Prediction project developed to understand how financial data and trend analysis can be integrated into a web application. The system takes historical stock data such as Open price, Close price, High, Low, and Volume and analyzes the data to identify basic market trends. Based on this analysis, the application predicts whether the stock is more likely to move in an upward or downward direction. The project is built using HTML, CSS, and JavaScript for the user interface, making it simple and easy to use. It helps users visualize stock behavior and understand how prediction logic can be applied to real-world financial data. This project was created for learning and academic purposes to gain practical experience in web development and data-driven applications.
